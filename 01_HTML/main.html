Dados dos Banco de Dados


Este projeto faz parte da avaliação da disciplina Desenvolvimento de Web, ministrada pelo professor Carlos Rafael Magalhães Fernandes, faculdade Impacta.
Originalmente ele é um trabalho em grupo, dividido em partes, mas, criei este projeto para treinar .js, .css, .html e BD.
O foco é juntar em um único lugar referências sobre Banco de Dados e seu mundo.
Para aqueles que se cadastrarem, poderão ter acesso a um quiz de multiplas respotas.
Agradeço meus pais, professores, amigos, colegas de trabalho e a todos em enviam, de forma gratuíta,materias sobre linguagens de programção, html, SQL entre outros temas.
